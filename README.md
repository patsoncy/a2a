# `a2a` == `凹凸凹` == `ALL TO ALL`

`a2a` is designed to echo pretty markdown in terminal,but far more.

## FEATURES

1. Convert any text to any type u like
2. rule supports `hot-swappable`

## EXAMPLE

``` javascript

var ata = require('ATA');

ata().use(rule).use(rule).parse('original.txt').output('parsed.txt');

```
